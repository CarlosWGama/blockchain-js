# Exemplo Blockchain em JavaScript

Uma simples Blockchain criada em JavaScript.

### JavaScript Puro
```
git clone https://github.com/CarlosWGama/blockchain-js/
cd blockchain-js
npm install
node index.js
```

### Rodando numa API (Express)
```
git clone https://github.com/CarlosWGama/blockchain-js/
cd blockchain-js
git checkout api
npm install
node index.js
```
